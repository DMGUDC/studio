# **App Name**: XChef Frontend

## Core Features:

- User Authentication: Implement login and authentication flow as per the backend API documentation. Note: currently only Google Fonts are supported.
- Dashboard: Develop the dashboard with access based on user roles (Gerente, Mesero, Cocinero).
- Table Management: Create visual table layouts using an HTML5 canvas, with drag-and-drop functionality.
- Order Management: Implement the multi-table order management system with order tracking, splitting bills and adding special requests.
- Inventory Management UI: Develop a user interface to show Inventory data. Inventory information is provided from the backend.
- Kitchen Display System (KDS): Implement a KDS to display orders and track preparation status using the states that are described in the documentation
- Financial Reporting: Display financial reports, COGS calculation, and profit margin analysis from the backend.

## Style Guidelines:

- Primary color: Dark slate blue (#37474F) for a professional and reliable feel.
- Background color: Light gray (#ECEFF1) to provide a clean and modern backdrop.
- Accent color: Amber (#FFB300) to highlight important elements and calls to action.
- Headline font: 'Poppins', a geometric sans-serif font for headlines. Body font: 'PT Sans', a humanist sans-serif font for body text
- Use simple, modern icons from a consistent icon set (e.g., Material Icons) to represent various functionalities.
- Design a responsive layout that adapts to different screen sizes, ensuring a consistent user experience across devices.
- Incorporate subtle transitions and animations to enhance user interaction and provide visual feedback (e.g., button hovers, loading spinners).
